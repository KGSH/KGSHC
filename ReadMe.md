## 오픈소스 결정 이유
보안을 신경쓰지 않고 오픈소스로 결정한 것은 다음과 같은 이유가 있습니다.
 1. 보안의 취약점을 이용해 해킹해 보는 것은 게임고인의 실력향상에 도움이 될 수 있지 않을까 생각했습니다.
 2. 게임고인이 모두 참여할수 있는 개방적 프로젝트를 제작하고자 하였습니다.
 3. 웹과 DB에 관심있는 게임고인에게 도움이 되고자 하였습니다.
